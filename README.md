## Requisitos Previos

Asegúrate de tener instalados los siguientes requisitos previos antes de comenzar:

- Node.js
- MongoDB

## Instalación

Para instalar las dependencias del proyecto, ejecuta:

```bash
npm install
```

## Configuración

## Ejecución
Para ejecutar el Backend usa el comando: 
npm start
